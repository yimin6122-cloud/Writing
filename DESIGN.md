# 灵感音域 — 线框图 & 设计规范 v1.0

## 一、页面结构

```
/ (首页)
├── Hero 大图区 (16:9, 可替换背景图)
├── 世界分类卡片网格 (9张)
└── 底部: 我的场景 + 论坛入口

/world (沉浸世界页)
├── 顶部: 返回按钮 + 场景标签栏
├── 左侧: 全屏视觉区 (背景图 + 粒子动画)
└── 右侧: 侧栏面板
    ├── [播放器] [歌单] [白噪音] [写作] 模式切换
    ├── 播放器面板 (迷你封面 + 进度条 + 控制按钮)
    ├── 歌单面板 (曲目列表 + 导入/上传/排序)
    ├── 白噪音面板 (15种音效网格 + 预设按钮)
    └── 写作面板 (编辑器 + 浮窗模式)
```

## 四、设计 Token

### 颜色
```
背景: #0a0a0f (最深), #111118 (卡片)
毛玻璃: rgba(255,255,255, 0.06/0.10/0.15)
边框: rgba(255,255,255, 0.08/0.15/0.25)
文字: #fff / rgba(255,255,255,0.85) / 0.55 / 0.35
主题色: 根据世界动态变化
```

### 间距
```
xs: 4px, sm: 8px, md: 12px, lg: 16px, xl: 20px, 2xl: 28px, 3xl: 40px
```

### 字号
```
caption: 11px, body-sm: 13px, body: 15px, body-lg: 17px
heading-sm: 20px, heading: 28px, hero: 48px
```

### 圆角
```
sm: 8px, md: 14px, lg: 22px, full: 9999px
```

### 动画
```
fast: 0.15s ease-out
normal: 0.3s ease-out
slow: 0.5s cubic-bezier(0.22, 0.61, 0.36, 1)
```

## 五、功能优先级 (MVP)

### P0 — 必须完美
- [ ] 首页 + 世界分类导航
- [ ] 场景标签切换
- [ ] 音乐播放器 (生成+上传+外部URL)
- [ ] 歌单管理 (排序/导入/复制/删除)
- [ ] 数据持久化 (IndexedDB)
- [ ] 毛玻璃 Apple 风格 UI

### P1 — 重要
- [ ] 白噪音混音器 (15种+预设+叠加)
- [ ] 写作功能 (编辑器+文档列表+浮窗)
- [ ] 场景创建/编辑/删除

### P2 — 锦上添花
- [ ] 粒子动画背景
- [ ] 音频可视化
- [ ] 快捷键
- [ ] 移动端适配

## 六、文件结构

```
index.html              ← 入口 (尽量精简)
css/styles.css          ← 全部样式 (按区块组织)
js/config.js            ← WORLD_TREE + 设计常量
js/db.js                ← IndexedDB 层
js/player.js            ← 音频播放引擎
js/noise.js             ← 白噪音引擎
js/writer.js            ← 写作功能
js/ui.js                ← UI 渲染 + 事件绑定
js/app.js               ← 主入口 + 状态管理 + 导航
```
